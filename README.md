# yaCmaera项目说明 #

http://www.yacamera.com  [ 已挂 ]
一个P2P流媒体平台，支持Internet上分享手机摄像头实时视频，通过浏览器直接查看。

## 技术路线 ##

* WebRTC技术，支持浏览器之间的数据传输
* 喷泉码技术，payload编码方式，支持多源点数据直接拼接
* HTML5技术，现代风格的用户体验，本地多媒体解码
* 手机视频源支持Android/iOS两大平台

这是一个非常酷的项目，几乎不需要任何的服务器就可以完成直播系统。
